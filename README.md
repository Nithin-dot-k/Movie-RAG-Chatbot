# 🎬 Movie RAG Chatbot

A local AI-powered movie assistant built with LangChain, Ollama, and ChromaDB.

## 🚀 Tech Stack

- Ollama (llama2 + mxbai-embed-large)
- LangChain
- ChromaDB
- Pandas

## ⚙️ Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Install and run Ollama locally
3. Run: `python main.py`
